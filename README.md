# QC-RNG

QC-RNG is a Geant4-based Monte Carlo simulation for the study of radioactive decays, gamma-ray detection and detector timing. The long-term goal is to simulate realistic radioactive sources (e.g. ^22Na), scintillation detectors and electronics to investigate coincidence measurements and random-number generation based on radioactive decay.

## Directory structure

```
QC_RNG/
|-- include/      C++ header files
|-- src/          C++ source files
|-- macros/       Geant4 macro files
|-- scripts/      Analysis scripts
|-- data/         Simulation output
|-- build/        Build directory (generated)
|-- CMakeLists.txt
`-- README.md
```

The `build` directory is generated by CMake and can safely be removed.

---

## Building

```bash
mkdir build
cd build

cmake ..
make -j
```

---

## Running

Interactive mode:

```bash
./QCRNG
```

Batch mode:

```bash
./QCRNG ../macros/run22Na.mac
```

Simulation output is written to

```
data/qcrng.root
```

---

## Configuration

Simulation parameters are controlled through Geant4 macro commands.

Example:

```tcl
/qcrng/detector/sourceSize 10 mm

/qcrng/detector/material LYSO
/qcrng/detector/threshold 50 keV
/qcrng/detector/energyResolution 0.08
/qcrng/detector/timingResolution 200 ps
/qcrng/source/activity 1 MBq

/run/initialize
/run/beamOn 100000
```

## ROOT output

Three TTrees are written.

### events

One entry per radioactive decay.

Typical quantities:

- eventID
- decayTime
